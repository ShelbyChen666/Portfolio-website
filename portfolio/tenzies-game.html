Tenzies Gameweb app

React, JavaScript, HTML & CSS

GitHub Repo Live version

Tenzies Game is a fun and interactive game that I created using React, JavaScript, CSS, and HTML. The game is based on the classic dice game called Tenzies, where the objective is to roll as many dice as possible with the same number on each dice. I designed and coded the entire user interface using CSS, creating a clean and visually pleasing layout that is easy to use and understand.

To build the game logic, I used React and specifically, the useState and useEffect hooks to manage the game state and to update the UI based on user interactions. I also implemented the game rules and logic, enabling players to roll the dices, hold the dices with same points, and re-roll the remaining dices until the same number on each dice. Additionally, the use of CSS Grid and Flexbox allowed for a highly customizable layout that is both visually appealing and responsive.

At the end of the game, I incorporated react-confetti to add a celebratory touch, where confetti explodes on the screen to celebrate the player's victory. Overall, Tenzies Game is a great example of my skills in React, CSS, and JavaScript development, showcasing my ability to create engaging and interactive games with dynamic user interfaces.
