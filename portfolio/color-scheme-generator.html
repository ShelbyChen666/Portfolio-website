Color Scheme
Generatorweb app

API integration, Responsive design, JavaScript, HTML & CSS

GitHub Repo Live version

The color scheme generator is a web application that allows users to select a base color and a color mode, and then generates a color scheme based on those choices. The application is built using HTML, CSS, and JavaScript, and utilizes thecolorapi.com API to retrieve color scheme data.

The application features a user-friendly interface with a large header, a color picker, a drop-down menu for selecting the color mode, and a "Get Color Scheme" button. Once the user selects their color and mode, the application generates a color scheme in the form of color boxes with the corresponding hex codes displayed below. The color boxes and hex codes are arranged in a grid for easy viewing.

The application is designed with responsive web design principles, ensuring a consistent and enjoyable user experience across devices. The interface adapts seamlessly to different screen sizes and resolutions, making it accessible and easy to use on any device.
