Personal DashboardChrome Extension

API integration, Async JS, JavaScript, HTML & CSS

GitHub Repo Live version

Personal Dashboard is a Chrome extension that offers a customizable dashboard on each newly opened Chrome tab. It retrieve real-time information about the user's location such as date, time, and weather conditions from OpenWeather API and it also comes with lastest market price of Bitcoin from CoinGecko API. Additionally, the Unsplash API provides an attractive and dynamically changing background image each time the user opens a new tab.

The app is built using HTML, CSS, and JavaScript, with special attention given to async JS practices for improving performance and optimizing data retrieval from APIs. The extension is easy to install and provides a convenient and personalized experience for users who frequently use Chrome.

Personal Dashboard is a smooth and functional extension that improves the user's browsing experience. It offers real-time and pertinent information that keeps users informed about the current weather and time, while also providing visually pleasing background images that change with each new tab opening.
