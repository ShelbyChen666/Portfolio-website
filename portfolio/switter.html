SwitterWeb Application

Vanila JS, DOM manipulation, HTML & CSS

GitHub Repo Live version

Switter is a Twitter-like web application that allows users to post tweets, reply to tweets, like and retweet tweets. The application is built using HTML, CSS, and Vanila JavaScript. The main functionalities of the application include:

  • Posting tweets:
    Users can type a tweet in the input area and click the "Tweet" button to post it. Each tweet is displayed with the user's avatar, username, and the tweet's content.
  • Replying to tweets:
    Users can see the replies of tweet by clicking the reply icon on the tweet. The replies and reply input area are then displayed, and the user can type their reply and post it by clicking the "Reply" button. JavaScript is used to toggle the visibility of the reply input and other replies of the tweet.
  • Liking and retweeting tweets:
    Users can like and retweet tweets by clicking on the heart and retweet icons, respectively. Likes and retweets are implemented using JavaScript to update the UI. The like icon and the retweet icon on each tweet are clickable and can toggle their states.
  • Unique tweet IDs:
    Each tweet is assigned a unique ID using the uuid library for generating unique IDs.

The application was built with Vanila JavaScript to handle user interactions with the UI. The JavaScript code is written in a modular and organized way, with separate functions for each functionality. The CSS code provides a clean and modern look for the application. Overall, this project demonstrates the use of HTML, CSS, and JavaScript to build a functional and interactive web application.
