War Card Gameweb app

API integration, DOM manipulation, JavaScript

GitHub Repo Live version

This project is a web-based implementation of the popular card game War, created using HTML, CSS, and JavaScript. The game is played with a standard deck of 52 cards, and the goal is to collect all the cards by winning "battles" against the computer. The project uses an API to retrieve a shuffled deck of cards, which is then split between the player and the computer.

The HTML and CSS files are used to create the game layout and style, while the JavaScript code handles game logic and user interaction. The JavaScript file contains the game logic, which determines the winner of each battle and keeps track of the score. The use of the DOM manipulation to dynamically update the game board and user interface based on the current game state.
